# Java 계산기 프로그램

Java로 만든 콘솔 계산기 프로그램

## 실행 방법

1. IntelliJ에서 프로젝트를 엽니다.
2. `App.java`의 `main` 메서드를 실행합니다.
3. 콘솔 안내에 따라 입력합니다.

## 기능

### 사칙연산
- 양의 정수 2개와 사칙연산 기호(+, -, *, /)를 입력받아 계산합니다.
- `remove` 입력: 가장 먼저 저장된 결과를 삭제합니다.
- `inquiry` 입력: 저장된 모든 결과를 조회합니다.
- `exit` 입력: 프로그램을 종료합니다.

### 원의 넓이 계산
- 반지름을 입력받아 원의 넓이를 계산합니다.
- `remove` 입력: 가장 먼저 저장된 결과를 삭제합니다.
- `inquiry` 입력: 저장된 모든 결과를 조회합니다.

## 사용 명령어

| 명령어 | 설명 |
|---|---|
| `calc` | 사칙연산 모드로 진입 |
| `circle` | 원의 넓이 계산 모드로 진입 |
| `remove` | 가장 먼저 저장된 사칙연산 결과 삭제 |
| `inquiry` | 저장된 결과 전체 조회 |
| `exit` | 프로그램 종료 |
